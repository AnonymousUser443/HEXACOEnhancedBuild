- **降维映射算法**：逻辑斯蒂回归，HEXACO → MBTI 四维度连续概率光谱
- **三层前端展示**：特质雷达图 → MBTI 光谱 + 九型标签 → 心灵剧场交互 UI

## 四、Sprint 2 — 核心评分引擎（已完成）

### 新增文件：`scoringEngine.js`

**核心功能模块**：

| 函数 | 功能 | 说明 |
|:-----|:-----|:-----|
| `applyReverseScoring` | 反向计分转换 | 5→1, 4→2, 3→3, 2→4, 1→5 |
| `calculateFacetScores` | 子面得分计算 | 含每题原始值/计分后值/是否反向 |
| `calculateDimensionScores` | 维度得分计算 | 原始分 + T分数 + z分数 + 百分位 |
| `calculatePercentile` | 百分位计算 | 正态分布累积分布函数（CDF）查表 |
| `mapToMBTI` | HEXACO→MBTI 映射 | 逻辑斯蒂回归，输出四字母类型 |
| `analyzeResponseTimes` | 响应时间分析 | 检测过快/过慢作答模式 |
| `calculateValidityIndex` | 效度指数计算 | 综合响应时间与一致性评分 |
| `calculateFullResult` | 完整结果计算 | 整合以上所有模块 |

### T 分数常模参数（HEXACO-CN）

| 维度 | 均值 | 标准差 |
|:-----|:-----|:-------|
| H（诚实-谦逊） | 3.5 | 0.55 |
| E（情绪性） | 3.2 | 0.60 |
| X（外向性） | 3.3 | 0.65 |
| A（宜人性） | 3.6 | 0.50 |
| C（严谨性） | 3.4 | 0.55 |
| O（开放性） | 3.5 | 0.60 |

公式：`T分数 = 50 + 10 × (原始分 - 均值) / 标准差`

### MBTI 逻辑斯蒂回归权重

- **EI**：X（外向性）为主导权重，E（情绪性）为负向调节
- **SN**：O（开放性）为主导权重，C（严谨性）为负向调节
- **TF**：A（宜人性）为主导权重，E（情绪性）为正向调节
- **JP**：C（严谨性）为主导权重，O（开放性）为负向调节
