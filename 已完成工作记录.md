## 一、设计阶段

### 文件：`多维人格交叉量表与网页展示系统设计.md`

一份完整的设计报告，涵盖：

- **HEXACO-CN 量表设计**：6 维度（H/E/X/A/C/O）× 4 子面，共 24 个子面。结合中国本土文化修正（情境化题干、中庸倾向适配）。
- **多维交叉验证架构**：
  - 荣格八维（John Beebe 心灵舞台模型）：阳面 4 功能（英雄/父母/孩童/阿尼玛）+ 阴面 4 功能（敌对/批评/盲点/恶魔）
  - 社会人格学（Socionics Model A）：Ego/Super-Ego/Id/Super-Id 四区块 + PoLR 最痛点检测
  - 九型人格：核心动机与恐惧辨析
- **防作弊机制**：情境判断测验（SJT）+ 响应时间（RT）采集 + 机器学习异常检测
- **降维映射算法**：逻辑斯蒂回归，HEXACO → MBTI 四维度连续概率光谱
- **三层前端展示**：特质雷达图 → MBTI 光谱 + 九型标签 → 心灵剧场交互 UI

### 数据库模型


## 四、Sprint 2 — 核心评分引擎（已完成）

### 新增文件：`scoringEngine.js`

**核心功能模块**：

| 函数 | 功能 | 说明 |
|:-----|:-----|:-----|
| `applyReverseScoring` | 反向计分转换 | 5→1, 4→2, 3→3, 2→4, 1→5 |
| `calculateFacetScores` | 子面得分计算 | 含每题原始值/计分后值/是否反向 |
| `calculateDimensionScores` | 维度得分计算 | 原始分 + T分数 + z分数 + 百分位 |
| `calculatePercentile` | 百分位计算 | 正态分布累积分布函数（CDF）查表 |
| `mapToMBTI` | HEXACO→MBTI 映射 | 逻辑斯蒂回归，输出四字母类型 |
| `analyzeResponseTimes` | 响应时间分析 | 检测过快/过慢作答模式 |
| `calculateValidityIndex` | 效度指数计算 | 综合响应时间与一致性评分 |
| `calculateFullResult` | 完整结果计算 | 整合以上所有模块 |

### T 分数常模参数（HEXACO-CN）

| 维度 | 均值 | 标准差 |
|:-----|:-----|:-------|
| H（诚实-谦逊） | 3.5 | 0.55 |
| E（情绪性） | 3.2 | 0.60 |
| X（外向性） | 3.3 | 0.65 |
| A（宜人性） | 3.6 | 0.50 |
| C（严谨性） | 3.4 | 0.55 |
| O（开放性） | 3.5 | 0.60 |

公式：`T分数 = 50 + 10 × (原始分 - 均值) / 标准差`

### MBTI 逻辑斯蒂回归权重

- **EI**：X（外向性）为主导权重，E（情绪性）为负向调节
- **SN**：O（开放性）为主导权重，C（严谨性）为负向调节
- **TF**：A（宜人性）为主导权重，E（情绪性）为正向调节
- **JP**：C（严谨性）为主导权重，O（开放性）为负向调节

### API 返回结构增强

`/api/sessions/:id/result` 现在返回：

```json
{
  "hexaco": [6个维度对象，含 rawScore/tScore/zScore/percentile/facets],
  "facets": [24个子面对象，含每题明细],
  "mbti": {
    "EI": {"type": "E", "probability": 0.92, "percent": 92},
    "SN": {"type": "N", "probability": 0.97, "percent": 97},
    "TF": {"type": "F", "probability": 1.0, "percent": 100},
    "JP": {"type": "J", "probability": 0.86, "percent": 86},
    "type": "ENFJ"
  },
  "validity": {
    "validityIndex": 1.0,
    "responseTimeAnalysis": {...},
    "reverseItemRatio": 0.48
  },
  "answeredCount": 96
}
```
